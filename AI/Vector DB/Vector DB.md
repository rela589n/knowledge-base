**Vector DB** is a database that stores data along with [[Embedding Vector|Vectors]],
optimized for similar [[Embedding Vector|Vector]] search.

> It's like [[Full Text Search]] which can <u>search by meaning</u>,
> 	not just words or synonyms.

These [[Embedding Vector|Vectors]] themselves are not sent to the LLM.
	The underlying data is.
